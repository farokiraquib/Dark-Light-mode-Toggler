# Dark/Light Mode Toggler 

A clean and interactive web application that allows users to switch between a dark and light theme. This project demonstrates dynamic DOM manipulation by updating both the visual styling of the page and the user interface text based on the current active state.

 **Live Demo:** [View the app here](https://modetogglerr.netlify.app/)

##  Features

* **Seamless Theme Switching:** Instantly swaps the page's color palette between dark and light modes.
* **Dynamic UI Updates:** The toggle button is context-aware. It updates its text automatically to read "Toggle to Light Mode" or "Toggle to Dark Mode" depending on the currently active theme.
* **Responsive Design:** Clean layout that looks great on both desktop and mobile screens.

##  Technologies Used

* **HTML5:** For the semantic structure of the page and button elements.
* **CSS3:** For styling the light and dark themes (likely utilizing CSS classes or variables for smooth transitions).
* **JavaScript:** For event listeners, checking the current theme state, and dynamically updating the text content of the DOM elements.
